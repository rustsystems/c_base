MLVS

Machine Learning Verification Service

Why MLVS?

No need to worry about your system configuration, operating system, dependencies installation etc

Give your network and property file - MLVS will verify for you

How?

Click on "Property verification" under "Our services"

Give a network model and a property file and a timeout value

Click on any tool to verify your network model for the given property with the given timeout value

Output after a run

Safe : Network is safe for the given property

Unsafe : Network is not safe for the given property

In this case, MLVS also displays falsifying inputs for which network is unsafe for the given property

Network model

Please provide your network model in Open Neural Network Exchange(.onnx) file format

Please clike here to know more

Property file

Please provide your property in .vnnlib fileformat

Please clike here to know more

Timeout value

If no timeout value is given, 20 sec is taken as default value
